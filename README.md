🧠 AI Quiz Generator
An AI-powered quiz generator that creates personalized multiple-choice quizzes on any topic you choose — right in your browser with Streamlit!


🚀 Features
✅ Generate multiple-choice questions (MCQs) using Google Gemini (Generative AI)
✅ Select topic & difficulty (Easy, Medium, Hard)
✅ Adjustable number of questions (1–10)
✅ Instant scoring and detailed feedback
✅ Caching to avoid repeated API calls
✅ Clean, responsive UI built with Streamlit


⚙️ Tech Stack
Python

Streamlit (Frontend)

LangChain (Prompt orchestration)

Google Gemini API (langchain-google-genai)

Pydantic (Output parsing)

dotenv (Environment variable management)
